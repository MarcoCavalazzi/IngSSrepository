User Interface


From this interface it is possible to send a sequence of move commands (forward, backward, left, right, stop) to the robot and see a virtual representation of the result on the right side of the page. While running, the robot will record the whole sequence of commands until the user decides to terminate the learning phase.

After the termination of the learning phase, the user can tell the robot to enter the autonomous phase through the homonymous button in the GUI and choose a 'direct' or a 'reverse' mode. During this phase the robot autonomously executes the sequence of moves it has learned, by complementing each move (e.g. forward->backward) if the selected mode is reverse.


During the autonomous phase, the robot is able to execute (as soon as the code allows it to) a stop command sent by the user (through the homonymous button in the GUI). If an obstacle is detected the robot executes an alternative behavior (in term of moves) in order to go around the object and follow the track again.
